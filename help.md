# Help

## tmux

prefix: `Ctrl + a`

### セッション

| キー         | 動作                 |
| ------------ | -------------------- |
| `tm`         | tmux 起動            |
| `tmn <name>` | 新規セッション作成   |
| `tma <name>` | セッションにアタッチ |
| `tmk <name>` | セッション削除       |
| `tmka`       | 全セッション削除     |
| `<pre> + d`  | デタッチ             |
| `<pre> + s`  | セッション一覧       |

### ウィンドウ

| キー               | 動作                 |
| ------------------ | -------------------- |
| `<pre> + c`        | 新規ウィンドウ       |
| `<pre> + n/p`      | 次/前のウィンドウ    |
| `<pre> + <number>` | ウィンドウ番号で移動 |
| `<pre> + &`        | ウィンドウ削除       |

### ペイン

| キー              | 動作               |
| ----------------- | ------------------ |
| `<pre> + \|`      | 縦分割             |
| `<pre> + -`       | 横分割             |
| `<pre> + h/j/k/l` | ペイン移動         |
| `<pre> + E`       | ペイン最大化トグル |
| `<pre> + x`       | ペイン削除         |

### レイアウト

`<pre> + L` でメニュー表示、数字キーまたは矢印キーで選択

| キー        | 動作               |
| ----------- | ------------------ |
| `<pre> + L` | レイアウトメニュー |
| `2`         | 縦分割 2ペイン     |
| `3`         | 縦分割 3ペイン     |
| `4`         | 縦分割 4ペイン     |
| `5`         | 横分割 2ペイン     |
| `6`         | 横分割 3ペイン     |
| `7`         | グリッド 4ペイン   |

### コピーモード

| キー        | 動作                   |
| ----------- | ---------------------- |
| `<pre> + [` | コピーモード開始       |
| `v`         | 選択開始 (vi)          |
| `y`         | コピー & 終了 (pbcopy) |
| `q`         | コピーモード終了       |

### その他

| キー        | 動作         |
| ----------- | ------------ |
| `<pre> + r` | 設定リロード |
